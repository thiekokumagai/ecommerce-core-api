# Plan 06-02: Formulários e Integração no Painel Administrativo (Frontend)

## Objective
Conectar a página `/configuracoes` e seus formulários no painel `admin-wonderland` com a nova API de configurações reais do backend (`ecommerce-core-api`), adicionando o carrossel premium de banners e busca de CEP automática.

## Tasks

<task id="1" title="Criar serviços e hooks de integração de API">
<read_first>
- src/services/product.service.ts (exemplo de padrão de serviços)
</read_first>

<action>
1. Criar `src/services/settings.service.ts`:
   - Implementar chamadas HTTP `getSettings()`, `updateSettings(data)` e `uploadMedia(file)`.
2. Criar hook `src/hooks/useSettings.ts`:
   - Configurar query `useQuery({ queryKey: ["settings"], queryFn: getSettings })` e mutação `useMutation` para persistir dados com invalidação automática de cache.
</action>
</task>
